Grouping Matrix Data

To group data in a matrix view, drag-and-drop one or more column headers to the space just above the matrix, which is labeled "Drag a column header and drop it here to group by that column".

When you drag-and-drop multiple column headers, you create a primary grouping and nested sub-groupings of the data based on the order of the headers.

You can change the primary grouping and order of the sub-groupings using drag-and-drop within these selections.

To undo a column grouping, click the "x" in the corner of the dropped header.
