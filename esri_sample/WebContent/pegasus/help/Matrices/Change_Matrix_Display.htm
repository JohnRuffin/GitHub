Changing Matrix Display

You can change matrix display settings in two ways:

  • Changing Application Settings for Products

  • Changing Matrix Settings

Changing Application Settings for Products

How to change the Products included in Matrix displays

  1. On the Applications Settings toolbar, click

  2. In the Volume Groups section, click the radio button of the group whose products you want to select from to include in all matrices, the Lane and Leg Detail, and the Market View Detail. You can display products from only one product group at a time.

  3. Click to expand the selected group's product listing.

  4. Select the checkbox of each product you want to include in the displays. Clear the checkbox of any product you don't want to include.

  5. Click Save.

 

Changing Matrix Field Display Settings

How to change Matrix Settings

  1. Open the desired matrix window.

  3. The matrix settings drawer window displays.

  4. Select or clear the checkboxes of the matrix display options as desired.

  5. Close the matrix settings window.

 

Matrix Display Options

Matrices have basically three types of matrix display settings, which are available in different combinations with different options depending on the matrix. The types and options are:

  • Show Columns section

    Individual column checkboxes

    A list of columns particular to each matrix. Select the checkbox to display, clear to hide.

    All Local Columns

    Select checkbox to display all columns in a matrix with specific time values, or values that can change according to time zone such as days (for example, M-S) or weeks (for example, Wk1-4), in local time. (L) displays in the column headings.

    All Zulu Columns

    Select checkbox to display all columns in a matrix with specific time values, or values that can change according to time zone such as days (for example, M-S) or weeks (for example, Wk1-4), in Zulu time. (Z) displays in the column headings.

    All Weight Columns

    Select checkbox to display all columns related to weight.

    All Cube Columns

    Select checkbox to display all columns related to cubic volume.

    All Pieces Columns

    Select checkbox to display all columns with piece counts.

     

     

  • Show Totals section

    Totals for each break

    When you group matrix data, select checkbox to display totals for each group and sub-group break.

    Grand totals

    Select checkbox to display grand totals for all columns with quantity data.

     

     

  • Set default week for Freq Wk column section

    Default week

    Select a Default week 1-4 as the default for the Freq Wk column.
