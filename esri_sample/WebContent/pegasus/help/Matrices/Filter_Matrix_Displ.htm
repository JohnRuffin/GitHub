Filtering a Matrix Display

You can filter a matrix display to include or exclude rows with certain values or ranges of values in a specific column. You can filter by as many of the available columns as needed.

This applies to all Pegasus matrix windows - Network, Schedule Overlay, Schedule, Volume Utilization, and Allocation.

How to set a filter

  1. Click  in the heading of the column you want to filter on. The filter window displays.

Matrix Filter Window

  1. In the first field, select from the drop-down list:

    • Is equal to

    • Is not equal to

    • Starts with

    • Contains

    • Does not contain

    • Ends with

  1. Type a value in the entry field to the right.

  2. If you want to filter using a secondary value:

    •  In the next field select And or Or from the drop-down list.

    • Then in the next field select a value from the drop-down list:

      • Is equal to

      • Is not equal to

      • Starts with

      • Contains

      • Does not contain

      • Ends with

    • Type a value in the entry field to the right.

  1. Click Filter to filter the rows by the specified value(s). Or click Clear to clear the filter.

  2. In the heading of each column with a filter, the icon changes to a white background .
