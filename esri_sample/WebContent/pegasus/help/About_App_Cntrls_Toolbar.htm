About the Pegasus Application

The Application Controls toolbar enables the planner to select the Plan and Schedule to work with. It also enables the planner to specify certain application-wide controls and to define Favorites.

Note: Some of the application controls are duplicated on the toolbars on individual windows. Where there is a conflict, the setting on the individual window toolbar overrides the application-wide setting.

Application Controls Toolbar

Select a Plan

  1. Click the Plan drop-down list

  2. Select the Plan to work with from the drop-down list. Access security by user/user group by plan type applies.

  3. List closes

 

Refresh Plan drop-down list

  1. To the right of the Plan drop-down list, click

  2. The Plan drop-down list refreshes to current data

 

Select a Schedule

  1. Click the Schedule drop-down list

  2. Select the Schedule

  3. List closes

  2. Empty mode analysis window displays

Open a different window

  2. Click the window to display

Change application settings

    1. Click

    2. Click radio buttons to Set effective days (MTWTFSS or MTWRFSN) and Set default time (Local or Zulu) to display by default

    3. Click radio buttons to Set default units for display (Weight - Pounds or Kilograms and Length - Miles or Kilometers)

    4. Click a radio button to specify default to Open plan on start-up (Open last used or Let me choose)

    5. Click a radio button to select a Volume Group, where you can specify which products are displayed in all matrices, the Lane and Leg Detail, and the Market View Detail. You can display products from only one Volume Group at a time.

    6. To see what products are in a volume group, expand the group by clicking .

    7. By default, all products in a group are displayed. To turn off the display of any product, clear the product checkbox.

    8. Click Save (or Cancel)
