Refreshing the display

There are a few ways to refresh the display:

Click

Use this option to refresh the window to match your query. If the window that you're refreshing has a default favorite, Pegasus applies any favorite settings.

Click , then Clear Favorite

Use this option to refresh the window and display system default settings. Pegasus clears any favorites applied to the window.

Click , then click a window name

Use this option to reopen a window that you have closed.

Click window tab at the bottom of the application window

Use this option to reopen a window that you have minimized.
