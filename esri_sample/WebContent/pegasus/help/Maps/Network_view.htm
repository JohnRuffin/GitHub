About Network View

 

The Network View of the map shows the lanes that match your Network search criteria using these symbols:

Circles and diamonds

represent location type: origins, destinations, origins and destinations, or transits

Line colors

represent lane direction: incoming, outgoing, or both

Line styles

represent mode: fly or truck

Broken red lines

represent connectivity issues

Numbers on lines

indicate that a line represents multiple lanes going between the same two points:

  • Lines that represent multiple lanes are always solid, regardless of the mode employed.

  • If the number is enclosed in a square, then all lanes use the same mode.

  • If the number is enclosed in a circle, then the lanes use different modes.
